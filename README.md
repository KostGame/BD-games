# BD Games

BD Games — статический учебный сайт с интерактивными заданиями по анализу и проектированию баз данных.

## Ссылки

- GitHub Pages: `https://kostgame.github.io/BD-games/`
- Тест 1: `https://kostgame.github.io/BD-games/tests/db-basics.html`

## Что внутри

- `index.html` — главная страница со списком доступных тестов.
- `tests/db-basics.html` — первый интерактивный тест на 10 вопросов.
- `styles.css` — общие стили для сайта.

## Как добавлять новые тесты

1. Создайте новый HTML-файл в `tests/`, например `tests/new-topic.html`.
2. Используйте `styles.css`, чтобы сохранить единый внешний вид.
3. Добавьте ссылку на новый тест в `index.html`.
4. Обновите README и, если нужно, добавьте отдельный отчет в `reports/`.

## Публикация

Проект рассчитан на GitHub Pages из корня репозитория. Страница `index.html` должна открываться по адресу главной публикации, а тесты — по прямым ссылкам внутри папки `tests/`.
