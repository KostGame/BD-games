Как расширять проект

Добавляйте новый HTML-файл в tests/, включайте его в список на главной и обновляйте README.

Структура

У каждого теста может быть собственная страница, а стили можно переиспользовать из styles.css.

Публикация

GitHub Pages должен отдавать index.html из корня репозитория и файл теста по прямой ссылке.
